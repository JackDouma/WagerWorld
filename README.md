<h2>Hardware</h2>
<ul>
  <li>React</li>
  <li>Firebase</li>
  <li>Phaser</li>
  <li>Javascript</li>
  <li>Colyseus</li>
  <li>Cloudflare</li>
</ul>

<h2>How to Run in VSC</h2>
<p>Add private firebase.js and .env file to Client directory.</p>
<h5>In terminal:</h5>
<ul>
  <li>cd Client</li>
  <li>npm i</li>
  <li>npm run dev</li>
</ul>
<h5>In another terminal:</h5>
<ul>
  <li>cd Sever</li>
  <li>npm i</li>
  <li>node server.js</li>
</ul>
